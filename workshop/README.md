# DNA Trait Predictor Workshop
## Complete Teaching Package

### Workshop Structure
- **Duration**: 4 weeks (4 sessions, 2 hours each)
- **Level**: Beginner-friendly (basic Python knowledge)
- **Format**: Hands-on coding with live demos

### Contents
1. `curriculum/` - Week-by-week lesson plans
2. `starter_code/` - Templates for students
3. `solutions/` - Complete working code
4. `data/` - Training datasets and samples
5. `slides/` - Presentation materials outline

### Quick Start
```bash
pip install pandas scikit-learn numpy matplotlib
python solutions/dna_predictor_complete.py
```

### Learning Outcomes
Students will:
- Parse genetic data from CSV files
- Extract features from SNP markers
- Train 3 machine learning models
- Build a GUI application
- Understand genetics + AI intersection
